Tor Proxy With Hacking Tools

Using Tor as a proxy when using pentesting tools is an easy way to improve anonymity.
From the terminal, we'll install the proxy and start the serivce

sudo apt-get install tor sudo serivce tor start

Let's check that Tor proxy is running.

sudo netstat -plnt Proto Recv-Q Send-Q Local Address tcp 0 0 127.0.0.1:9050

Tor proxy is running on port 5050. The proxy can now be specified when using terminal tools. For example:

sudo wpscan --url example.com --proxy=socks5://127.0.0.1:9050

Graphical tools tend to have an option to add a proxy
